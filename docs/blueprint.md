# **App Name**: CityFind

## Core Features:

- City Search: Main page search bar for searching distributors in a specific city.
- Language Support: Language selection button for internationalization.
- Business Sign-Up/In: Sign-up and sign-in functionality for businesses.
- Category Browsing: Category page displaying various business categories (food, drinks, electronics, etc.).
- Member Dashboard: Member dashboard for managing product listings, details, and varieties. This dashboard has add and remove options for member business.
- Product Listing: Product display page with A-Z listing and search functionality for guest users.
- Proximity Search: Distributor search with proximity-based results and address display. Members appear who are located nearest to guest user, the listing display up to <=100km. Members can set location and adjust business types. Members should have search visibility depending on those conditions.

## Style Guidelines:

- Primary color: HSL 45, 100%, 50% - A vibrant gold (#FFDA63) to symbolize opportunity and success.
- Background color: HSL 45, 20%, 95% - A light cream (#F5F4F0), which complements the primary and offers a clean and modern backdrop.
- Accent color: HSL 15, 70%, 50% - A warm orange (#E07A5F) used sparingly for calls to action and highlights, conveying urgency and energy.
- Headline font: 'Playfair' (serif) for headlines and small amounts of text. Body font: 'PT Sans' (sans-serif) for all other body text.
- Clean, modern icons that represent categories, search, and user actions.
- A clean, card-based layout to focus attention. Focus on presenting data that will let users quickly determine the options nearest to them.
- Subtle animations when interacting with the page or switching to product varieties.
